# RABIES Preprocessing and Confound Correction Pipeline

This directory contains resting-state fMRI timeseries data that has been preprocessed and cleaned using RABIES (Rodent Automated Bold Improvement of EPI Sequences) pipeline version 0.5.5.
Desrosiers-Grégoire, G., Devenyi, G.A., Grandjean, J. et al. A standardized image processing and data quality platform for rodent fMRI. Nat Commun 15, 6708 (2024). https://doi.org/10.1038/s41467-024-50826-8
https://github.com/CoBrALab/RABIES

 
**RABIES was run seperately on each subject to generate a subject average from all 5 session to improve registrations**

## 1. Reference Template (`space-normative`)
All data in this folder designated with `space-normative` has been registered to the custom unbiased population average template included in this dataset (`derivatives/population_template/tpl-normative`). 
* **Target Registration Template:** 0.2mm isotropic population average (`tpl-normative_res-02_T2w.nii.gz`)
* **Functional Resampling Resolution:** 0.3mm isotropic

## 2. Anatomical and Functional Preprocessing
The initial preprocessing stage was run with the following key parameters:
* **Inhomogeneity Correction:** SyN-based bias field correction with Otsu thresholding (2) applied to both anatomical and bold images.
* **Head Motion Correction (HMC):** Optimized HMC applied. (Note: Slice-timing correction and despiking were not applied).
* **Spatial Normalization:** Registration to the common space template was performed using SyN. 
* **Fast Commonspace = False** This creates a subject average from all inputs - in this case each subjects 5 scans.
## 3. **CRITICAL METHODOLOGICAL NOTE: Manual Mask Replacement**
Following the initial RABIES preprocessing stage, the default functional brain masks generated by RABIES were replaced with ones generated in AFNI to enforce strict signal boundaries:
* The RABIES-generated `bold_mask` files were **manually replaced** with highly binary brain masks generated via AFNI.
* These strict AFNI masks can be found in the separate `derivatives/AFNI_masks/` directory.
* The confound correction stage detailed below was run *using these strict AFNI masks*.

## 4. Confound Correction 
The confound stage (`desc-clean_bold.nii.gz`) was executed with the following parameters to yield the final analysis-ready timeseries:
* **Image Scaling:** Grand mean scaling was applied to the timeseries.
* **Detrending:** First-order (linear) detrending.
* **Nuisance Regression:** The timeseries was orthogonalized against:
  * Global Signal (GSR)
  * 6 head motion parameters (`mot_6`)
* **Temporal Filtering:** A bandpass filter was applied retaining frequencies between **0.01 Hz and 0.1 Hz**.
* **Spatial Smoothing:** An isotropic spatial smoothing filter of **0.5 mm** was applied.
* **Frame Censoring:** While Framewise Displacement (FD) was calculated, explicit frame censoring (scrubbing) was *not* applied during this pipeline stage.
* **Edge Removal** 30 seconds was removed from each side of the timeseries to avoid introducing artefacts from filtering - timeseries is now 940 volumes/repitions instead of 1000
