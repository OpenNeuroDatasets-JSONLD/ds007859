# Longitudinal Growth Metrics

This directory contains growth tracking information for the subjects in this dataset. Metrics include daily body weight and weekly body and tail length

## Files Included
* **`daily_weights.csv`**: Contains daily body weight measurements (in grams) spanning from Postnatal Day 21 (P21) through until the day of the last scan (approx P91).
* **`weekly_lengths.csv`**: Contains weekly body and tail length measurements (in centimeters) tracking growth across the same developmental window.

## Measurement Methodology
* **Body Weight:** Measured daily using a standard laboratory scale to one decimal place.
* **Body Length:** Measured from the tip of the nose to the ischium, a posterior bony projection on the hip bone.
* **Tail Length:** Measured from the ischium to the tip of the tail.
